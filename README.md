# BunnyCure

Basic Spring Boot project scaffolding with domain, service, and web layers plus template and static assets.

## Structure

- `src/main/java/cl/bunnycure/config`: configuration placeholders
- `src/main/java/cl/bunnycure/domain`: domain models, enums, repositories
- `src/main/java/cl/bunnycure/service`: application services
- `src/main/java/cl/bunnycure/web`: controllers and DTOs
- `src/main/java/cl/bunnycure/exception`: exception types
- `src/main/resources/templates`: HTML templates
- `src/main/resources/static`: static assets

## Try It

```bat
mvnw.cmd test
mvnw.cmd spring-boot:run
```
