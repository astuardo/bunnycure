  ┌─────────────────────────────────────────────┐
  │ 1. Scheduler dispara sendDailyReminders()   │
  ├─────────────────────────────────────────────┤
  │ 2. Busca citas pendientes para HOY          │
  │    (notificationSent = false)               │
  ├─────────────────────────────────────────────┤
  │ 3. Para cada cita:                          │
  │    - Obtiene datos del cliente              │
  │    - Obtiene datos de la cita               │
  │    - Envía email de recordatorio            │
  │    - Envía WhatsApp (si disponible)         │
  ├─────────────────────────────────────────────┤
  │ 4. Marca: notificationSent = true           │
  │    (Previene duplicados)                    │
  ├─────────────────────────────────────────────┤
  │ 5. Registra en logs [REMINDER]              │
  └─────────────────────────────────────────────┘
